O Git é um sistema de controle de versão distribuído que permite aos desenvolvedores gerenciar, rastrear e colaborar em projetos de software, registrando alterações no código ao longo do tempo. Ele facilita o trabalho em equipe, possibilitando a criação de branches independentes e a fusão de alterações sem conflitos.

GitHub é uma plataforma baseada na web que utiliza o Git como núcleo, oferecendo recursos adicionais como repositórios remotos, ferramentas de colaboração, integração contínua, controle de acesso e interfaces amigáveis para gerenciar projetos de forma eficiente e compartilhada. Enquanto o Git é a ferramenta, o GitHub é o serviço que amplia suas capacidades.
